# MVP Quiz Service

MVP-платформа для проведения квизов в реальном времени.

## О проекте

Сервис позволяет организаторам создавать и проводить квизы, а участникам подключаться к активной сессии по коду комнаты и отвечать на вопросы синхронно с показом.

## Ключевые возможности

- Регистрация и вход для двух ролей: организатор и участник
- Конструктор квиза: категория, тайминг, правила прохождения
- Поддержка вопросов с текстом и изображениями
- Поддержка одиночного и множественного выбора
- Запуск сессии и подключение участников по коду комнаты
- Показ вопросов и прием ответов в режиме реального времени
- Автоматический подсчет баллов и формирование лидерборда
- Личные кабинеты с историей проведенных и пройденных квизов

## Технологии

- Frontend: Angular 19
- Backend: Node.js + Express
- База данных: SQLite
- Реальное время: Socket.IO

## Архитектура

- Клиентская часть: Angular-приложение для организатора и участника
- Серверная часть: REST API + WebSocket-события через Socket.IO
- Хранение данных: SQLite

## Сценарий работы

1. Организатор создает квиз и запускает сессию.
2. Сервис генерирует код комнаты.
3. Участники подключаются по коду.
4. Вопросы последовательно показываются всем участникам.
5. Ответы принимаются только в рамках активного времени вопроса.
6. По завершении формируется итоговый лидерборд.

## Запуск

Backend:

```bash
cd backend
npm install
npm run start
```

Frontend:

```bash
cd frontend
npm install
npm start
```

## Результат

Работоспособный MVP, который покрывает основной цикл проведения квиза: от создания до публикации результатов.

- Организатор создает и запускает квиз
- Участники проходят квиз в реальном времени
- Система формирует лидерборд и сохраняет результаты
